Reference: Bo Y C ,  Wang P ,  Zhang X . An asynchronously deep reservoir computing for predicting chaotic time series[J]. Applied Soft Computing, 2020, 95(6):106530. https://doi.org/10.1016/j.asoc.2020.106530
